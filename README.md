## Présentation  
Cette application permet de gérer un carnet de contacts : vous pouvez ajouter de nouveaux contacts, mettre à jour ou supprimer des contacts existants, et consulter la liste.  
Elle est conçue pour exercer la gestion de données locales, d’interface utilisateur, et l’organisation du code dans un cadre de projet.

## Fonctionnalités  
- Ajout d’un contact (nom, téléphone, e‑mail, autres champs selon configuration).  
- Modification d’un contact existant.  
- Suppression d’un contact.  
- Affichage de la liste des contacts.  
- Recherche/filtre (si implémenté).  
- Stockage local des données (fichier, base de données légère ou autre).  
- Interface utilisateur intuitive.

## Utilisation  
- Ouvrez l’application.  
- Utilisez l’option « Ajouter un contact » pour enregistrer un nouveau.  
- Sélectionnez un contact pour le modifier ou le supprimer.  
- Utilisez la recherche pour trouver un contact (si cette fonction est implémentée).  
- Vérifiez que les contacts restent enregistrés entre les sessions.
